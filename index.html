Hekate Online Launcher

Instructions:

  1. Put the Switch in RCM mode and connect it to your device.
  2. Press the 'Load Hekate' button.
  3. On the consent screen that appears, select 'APX' and hit confirm.
  4. If all goes well, the payload will launch!

FAQ:

  • This does NOT work on Windows due to a limitation in the Chrome implementation of WebUSB (and probably other reasons!)
  • This should work on Chrome, Edge, Samsung Internet, and Opera. Make sure you have the latest versions of the browser.
  • This thing is possible due to WebUSB.
  • On Linux, you might get an access denied error! If you do, you can try creating a file at /etc/udev/rules.d/50-switch.rules
    With the following contents:
    SUBSYSTEM=="usb", ATTR{idVendor}=="0955", MODE="0664", GROUP="plugdev"
  • This has been tested and appears to work on Linux, OSX, Android (unrooted), and Chromebooks. Your mileage may vary.
